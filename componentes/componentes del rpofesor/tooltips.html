Componente Tooltips

Este componente muestra información adicional al pasar el cursor sobre un elemento, siguiendo la metodología BEM.

Variante 1: Tooltip Estándar

Tooltip básico que aparece al pasar el cursor sobre un elemento.

Este es un tooltip estándar

Variante 2: Tooltip con Posiciones

Tooltips que aparecen en diferentes posiciones: arriba, derecha, abajo e izquierda.

Tooltip arriba
Tooltip derecha
Tooltip abajo
Tooltip izquierda

Variante 3: Tooltip con Colores

Tooltips con diferentes colores de fondo para diferentes tipos de información.

Tooltip primario
Tooltip secundario
Tooltip acento
Tooltip oscuro

Variante 4: Tooltip con Flechas

Tooltips con flechas que apuntan al elemento que los activa.

Tooltip con flecha
Tooltip con flecha arriba
Tooltip con flecha derecha
Tooltip con flecha abajo

Variante 5: Tooltip con Animación

Tooltips con diferentes efectos de animación al aparecer y desaparecer.

Tooltip con fade
Tooltip con scale
Tooltip con slide
Tooltip con bounce
